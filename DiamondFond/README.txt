Программа валидирует файл files/DiamondFond.xml используя files/DiamondFondXMLSchema.xsd,
разбирает файл DiamondFond.xml с помощью SAX, DOM или StAX парсера,
используя полученную информацию создает объект класса Pavilion,
выводит его текстовое представление на экран.
Преобразует DiamondFond.xml файл в DiamondFondTable.html файл.
